# Hotel Booking MERN App

## Tech Stack :-
- Nodejs
- Typescript
- ReactJs
- MongoDB
- Express js


## App Features :-
- Signup User
- Login User
- User Authentication
- User Authorization


## Frontend Setup :-
- npm create vite@latest
- client
- React,Typescript
- cd client
- npm install
- To run Frontend Project :-  npm run dev



## Backend Setup :-
- npm init -y
- npm install express
- npm install -D typescript ts-node-dev @types/node @types/express
- npx tsc --init
- npm install -D concurrently
